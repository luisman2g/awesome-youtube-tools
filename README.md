## Transcription & Subtitles

Tools for extracting, generating, and downloading transcripts and subtitles from YouTube videos.

| Tool | Description | Free? | Open Source? |
|------|-------------|-------|--------------|
| ⭐ [**YouTubeTranscriptFree**](https://youtubetranscriptfree.com) | Free YouTube transcript generator with timestamps. No login required, supports 100+ languages. Copy or download as TXT. | ✅ Free | — |
| [Tactiq](https://tactiq.io/tools/youtube-transcript) | Chrome extension for real-time transcript extraction during video playback. | Freemium | — |
| [NoteGPT](https://notegpt.io/youtube-transcript-generator) | YouTube transcript generator with AI note-taking features. | Freemium | — |
| [youtube-transcript-api](https://github.com/jdepoix/youtube-transcript-api) | Python library to fetch YouTube transcripts programmatically. | ✅ Free | ✅ |
| [Whisper](https://github.com/openai/whisper) | OpenAI's speech recognition model. Can transcribe any audio including YouTube videos without subtitles. | ✅ Free | ✅ |
| [Maestra](https://maestra.ai/tools/video-to-text/youtube-transcript-generator) | AI-powered transcription with translation and voiceover capabilities. | Freemium | — |
| [TubeTranscript](https://tubetranscript.com/en) | Simple YouTube video transcription tool with multi-language support. | ✅ Free | — |
| [Kome](https://kome.ai/tools/youtube-transcript-generator) | Fast YouTube transcript generator as part of a larger AI toolkit. | ✅ Free | — |

## Contributing

Contributions are welcome! Please:

1. Fork this repository
2. Add your tool in the appropriate category
3. Use the format: `[Tool Name](URL) | Description | Free?`
4. Submit a Pull Request

Please ensure your PR:
- Adds only one tool per PR
- Follows the existing format
- Includes a brief, accurate description
- Links to the official website
